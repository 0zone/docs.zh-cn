# dayname

## 功能

返回指定日期对应的星期名称。

参数为Date或者Datetime类型。

## 语法

```Haskell
VARCHAR DAYNAME(DATE)
```

## 示例

```Plain Text
MySQL > select dayname('2007-02-03 00:00:00');
+--------------------------------+
| dayname('2007-02-03 00:00:00') |
+--------------------------------+
| Saturday                       |
+--------------------------------+
```
